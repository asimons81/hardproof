# Support

Use GitHub Discussions for usage questions, workflow guidance, and compatibility experiences. Use GitHub Issues for reproducible defects and scoped feature requests. Use the private process in `SECURITY.md` for vulnerabilities.

Before requesting help, run:

```bash
hermes hardproof doctor
python scripts/inspect_hermes_api.py
```

Include Hardproof version, Hermes version, operating system, command surface, relevant sanitized output, and a minimal reproduction. Never post API keys, cookies, authorization headers, `.env` files, private prompts, or unredacted databases.

Support is community and maintainer best-effort until a published support policy supersedes this document.
