## Known Deficits From v0.3.1

These findings should be addressed before or during v0.4.0:

1. **Dead code:** `claim_workcell_task` exhaustion check (line 392) is unreachable — `authorize_workcell_retry` duplicates the check. Consider removing or documenting.
2. **Windows symlink test gaps:** Symlink-based escape detection tests are skipped on Windows. Evaluate adding conditional test infrastructure.
3. **Plugin.yaml packaging:** `plugin.yaml` is at the repo root but not included in the PyPI wheel. Evaluate whether Hermes needs it at runtime or only the entry point is sufficient.
4. **docs/check_docs.py:** Fix the false-positive absolute-path scanner that triggers on test/script code.
5. **SQLite concurrency testing:** No isolation test for active-child limit under concurrent scheduler calls.
